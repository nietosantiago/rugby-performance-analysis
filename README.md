# 🏉 Rugby Performance Analysis

Proyecto de Data Science aplicado al rugby para analizar el rendimiento de jugadores y equipos a partir de videos de partidos.

El objetivo es transformar datos de juego en métricas accionables para entrenadores y jugadores.

---

## 📊 Objetivo

Este proyecto permite analizar partidos de rugby y generar estadísticas como:

- Tackles
- Carries
- Rucks
- Lineouts
- Kicks

Además incluye:

- Métricas avanzadas
- Mapas de calor
- Análisis por jugador
- Análisis por equipo

---

## 🎯 Caso de uso

El sistema está pensado para:

- Equipos de rugby amateur y semi-profesionales
- Analistas de rendimiento
- Entrenadores
- Jugadores

Permite cargar videos de partidos y obtener análisis de rendimiento.

## 📈 Métricas analizadas

### Equipo

- Tackles totales
- Carries totales
- Rucks ganados
- Lineouts
- Kicks

### Jugadores

- Tackles por jugador
- Carries por jugador
- Participación en rucks
- Efectividad
